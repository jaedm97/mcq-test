This is a plugin to maintain easily a multiple choice question website.

== Description ==

Using this awesome WordPress plugin, you can easily maintain / manage your website as a multiple choice questin ( MCQ ) platform. There are many features added on this plugins.

### Multiple Choice Question (MCQ) by pluginbazar - Jaed Mosharraf

== Installation ==

1. Upload Multiple Choice Question (MCQ) to the /wp-content/plugins/ directory
2. Activate the plugin through the Plugins menu in WordPress
3. Now select from the Admin menu Woocommerce open close and edit your settings.
4. If you want to get all the features then Click Buy in the bottom of the page.
5. If you have inquiry then please contact. http://pluginbazar.ml/contact/
